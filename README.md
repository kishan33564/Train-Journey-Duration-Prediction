# Train Journey Duration Prediction

## Overview
A Machine Learning application that predicts train journey duration based on:

- Total Distance
- Number of Stops
- Starting Station
- Ending Station

## Features

- Interactive Streamlit Web App
- Real-time Prediction
- Actual vs Predicted Visualization
- Linear Regression Model
- MAE, RMSE and R² Evaluation

## Technologies

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Run

pip install -r requirements.txt

streamlit run app.py
